Справка по условиям поиска

Можно различить следующие несколько видов условий поиска:

Поиск подстроки

Данное условие предполагает ввод искомой подстроки в обычном текстовом поле. Например, под условие 'ст' попадут следующие строки:
  • 'станция'
  • 'аист'
  • 'остановка'

Поиск по интервалу дат

Данное условие предполагает ввод границ интервала дат в котором необходимо осуществлять поиск.

Поиск по числовому интервалу

Данное условие предполагает ввод границ интервала чисел в котором необходимо осуществлять поиск.
